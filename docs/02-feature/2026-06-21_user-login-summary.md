## 🎯 功能特性

### 登录流程
1. **自动登录**: 应用启动时自动检查并登录
2. **无感登录**: Token 有效期内自动保持登录状态
3. **Token 刷新**: Token 即将过期时自动刷新
4. **手动登录**: 点击头像可手动触发登录

### 个人信息管理
1. **头像上传**: 支持相册选择或拍照
2. **昵称修改**: 实时字符计数，最多20字
3. **性别设置**: 男/女/保密三选一
4. **生日设置**: 日期选择器
5. **个性签名**: 最多100字

### 用户体验
1. **未登录浏览**: 未登录也可浏览，部分功能受限
2. **新用户引导**: 首次登录引导完善信息
3. **数据缓存**: 用户信息本地缓存，快速展示
4. **友好提示**: 完善的错误提示和操作反馈

## 🚀 使用说明

### 用户操作流程

1. **首次使用**
   - 打开小程序自动登录
   - 新用户会收到完善信息引导

2. **修改个人信息**
   - 进入"我的"页面
   - 点击头像或"个人资料"
   - 修改信息后保存

3. **退出登录**
   - 进入"我的"页面
   - 滑到底部
   - 点击"退出登录"

### 开发调试
