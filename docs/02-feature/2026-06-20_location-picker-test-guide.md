## 🎯 使用说明

### 用户操作流程

1. **选择地点**
   - 在饮品详情页，点击"地点"卡片
   - 在打开的地图中搜索或选择位置
   - 点击"确定"返回

2. **查看地图预览**
   - 选择地点后自动显示地图预览
   - 可以看到地点名称和详细地址

3. **清除地点**
   - 点击地点卡片右侧的清除图标
   - 地点信息清空

4. **重新选择**
   - 点击地点卡片或地图预览
   - 重新打开地图选择器

### 保存记录

- 保存时位置信息自动包含在记录数据中
- location 对象包含：名称、地址、经纬度

## 🔧 技术实现要点

### 1. 使用 uni.chooseLocation()

```javascript
// 调用微信小程序原生地图选择器
const result = await uni.chooseLocation({
  success: (res) => {
    // res.name - 地点名称
    // res.address - 详细地址
    // res.latitude - 纬度
    // res.longitude - 经度
  }
})
```

### 2. 权限处理

```javascript
// 检查权限
const res = await uni.getSetting()
if (res.authSetting['scope.userLocation'] === false) {
  // 用户拒绝过，引导去设置
  uni.openSetting()
}
```

### 3. 地图组件

```vue
<map
  :latitude="record.latitude"
  :longitude="record.longitude"
  :markers="mapMarkers"
  :scale="16"
/>
```
