# 阿里云 OSS 对象存储费用详解

## 费用组成

阿里云 OSS 费用主要包括以下几部分：

### 1. 存储费用（按容量计费）

| 存储类型 | 价格（元/GB/月） | 适用场景 |
|---------|----------------|---------|
| **标准存储** | 0.12 元/GB/月 | 频繁访问的数据，如图片、视频 |
| **低频访问存储** | 0.08 元/GB/月 | 访问频率较低，最少存储 30 天 |
| **归档存储** | 0.03 元/GB/月 | 长期归档，最少存储 60 天 |
| **冷归档存储** | 0.01 元/GB/月 | 极低频率访问 |

### 2. 流量费用（按传输量计费）

| 流量类型 | 价格 |
|---------|------|
| **内网流出流量** | 免费 | OSS 与 ECS 之间同地域传输 |
| **外网流出流量** | 0.50 元/GB | 从 OSS 下载到客户端 |
| **CDN 回源流量** | 0.15 元/GB | 从 OSS 到 CDN |
| **传入流量** | 免费 | 上传到 OSS |

### 3. 请求费用（按次数计费）

| 请求类型 | 价格（元/万次） |
|---------|---------------|
| **PUT/POST 请求** | 0.01 元/万次 |
| **GET 请求** | 0.01 元/万次 |
| **删除请求** | 免费 |

### 4. 图片处理费用

| 处理类型 | 价格 |
|---------|------|
| **图片处理（缩放、裁剪、水印等）** | 0.025 元/千次 |
| **视频截帧** | 0.10 元/千次 |

## 💰 费用估算示例

### 示例：饮品记录小程序

**假设使用情况：**
- 每月上传 500 张图片
- 每张图片平均 500KB（压缩后）
- 每张图片被查看 10 次（包含抠图处理）

**月度费用计算：**

#### 1. 存储费用
```
500 张 × 0.5MB = 250MB = 0.25GB
0.25GB × 0.12元/GB = 0.03元
```
**≈ 0.03 元/月**

#### 2. 流量费用（外网流出）
```
500 张 × 10次查看 × 0.5MB = 2500MB = 2.5GB
2.5GB × 0.50元/GB = 1.25元
```
**≈ 1.25 元/月**

#### 3. 请求费用
```
上传：500次 PUT × 0.01元/万次 = 0.005元
下载：5000次 GET × 0.01元/万次 = 0.005元
```
**≈ 0.01 元/月**

#### 4. 总费用
```
存储：0.03元
流量：1.25元
请求：0.01元
总计：≈ 1.29 元/月
```

## 🔥 省钱技巧

### 1. 使用 CDN 加速

**开通阿里云 CDN：**
- OSS → CDN 回源流量：0.15元/GB（比直接访问便宜）
- CDN → 用户流量：0.24元/GB（国内加速）
- **总成本**：0.15 + 0.24 = 0.39元/GB
- **直接访问成本**：0.50元/GB
- **节省**：约 22%

### 2. 使用图片压缩

**OSS 图片处理功能：**
- 自动压缩图片质量
- 调整图片尺寸
- **效果**：可减少 50%-70% 的流量成本

### 3. 使用生命周期管理

**自动转存低频存储：**
- 设置规则：30天未访问自动转为低频存储
- 节省：存储费用从 0.12元/GB 降至 0.08元/GB
- 节省比例：33%

### 4. 购买资源包

**预付费资源包：**
- 40GB 存储包：约 9元/月（比按量 4.8元便宜，但够用）
- 100GB 流量包：约 35元（比按量 50元便宜）
- **节省**：约 30%

## 📝 快速开通 OSS

### 开通步骤

1. **登录阿里云控制台**
   https://oss.console.aliyun.com/

2. **创建 Bucket**
   - Bucket 名称：`bty-food-diary`
   - 地域：华东1（杭州）或离你最近的区域
   - 存储类型：标准存储
   - 读写权限：公共读（图片需要公开访问）

3. **配置 CORS**（小程序访问需要）
   ```
   允许来源：*
   允许方法：GET, POST, PUT
   允许头：*
   暴露头：ETag, x-oss-request-id
   缓存时间：3600
   ```

4. **获取 AccessKey**
   - 使用已创建的 RAM 用户 AccessKey
   - AccessKey ID: `YOUR_ACCESS_KEY_ID`
   - AccessKey Secret: `YOUR_ACCESS_KEY_SECRET`

5. **安装 OSS SDK**
   ```xml
   <dependency>
       <groupId>com.aliyun.oss</groupId>
       <artifactId>aliyun-sdk-oss</artifactId>
       <version>3.17.4</version>
   </dependency>
   ```
