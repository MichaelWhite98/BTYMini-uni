# 百淘云小程序 UI 设计优化方案

## 设计系统概览

### 配色系统

| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | #11998e | 主色调，按钮、图标、强调文字 |
| Primary Light | #38ef7d | 渐变终点色，状态指示 |
| Primary Container | #e8f5f3 | 主色容器背景，选中状态 |
| Secondary | #f57c00 | 次要色，警示、辅助信息 |
| Background | #fafafa | 页面背景 |
| Surface | #ffffff | 卡片、容器背景 |
| On Surface | #1a1a1a | 主要文字颜色 |
| On Surface Variant | #666666 | 次要文字颜色 |
| Error | #e53935 | 错误状态 |
| Success | #38ef7d | 成功状态 |
| Warning | #f57c00 | 警告状态 |

### 间距系统

| 名称 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 最小间距 |
| sm | 8px | 紧凑间距 |
| md | 16px | 标准间距 |
| lg | 24px | 宽松间距 |
| xl | 32px | 大间距 |
| container-padding | 20px | 页面左右边距 |

### 圆角系统

| 名称 | 值 | 用途 |
|------|-----|------|
| sm | 8px | 小元素 |
| default | 12px | 按钮、输入框 |
| md | 16px | 中型卡片 |
| lg | 20px | 大型卡片 |
| xl | 24px | 容器卡片 |
| 2xl | 28px | 页面级卡片 |
| full | 9999px | 圆形、胶囊 |

### 字体系统

| 名称 | 大小 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| Display LG | 36px | 44px | Bold | 大数字统计 |
| Headline LG | 28px | 36px | Bold | 页面标题 |
| Title MD | 18px | 24px | Semibold | 卡片标题、按钮文字 |
| Body LG | 16px | 24px | Regular | 主要内容文字 |
| Body SM | 14px | 20px | Regular | 辅助说明文字 |
| Label Caps | 12px | 16px | Bold | 标签、提示文字 |

### 阴影系统

| 名称 | 值 | 用途 |
|------|-----|------|
| xs | 0 2px 8px rgba(17, 153, 142, 0.04) | 轻微阴影 |
| sm | 0 4px 12px rgba(17, 153, 142, 0.06) | 小元素 |
| card | 0 4px 20px rgba(17, 153, 142, 0.08) | 卡片 |
| lg | 0 8px 24px rgba(17, 153, 142, 0.12) | 浮动元素 |
| fab | 0 8px 24px rgba(17, 153, 142, 0.3) | 浮动按钮 |

- 多条记录时右上角显示数量徽章
- 统计卡片数字使用渐变文字效果
- 底部导航栏使用毛玻璃效果 (`backdrop-filter: blur(20px)`)

## 组件规范

### 按钮尺寸

| 类型 | 尺寸 | 使用场景 |
|------|------|----------|
| 小型 | 32px | 编辑按钮、小图标按钮 |
| 标准 | 40px | 导航栏按钮、列表项按钮 |
| 中型 | 48px | 底部导航按钮 |
| 大型 | 56px | FAB 浮动按钮 |

### 导航栏

```scss
.app-header {
  position: sticky;
  top: 0;
  z-index: 50;
  height: calc(var(--status-bar-height) + 44px); // 适配灵动岛
  padding: 0 20px;
  background: var(--background);
  // 或毛玻璃效果
  backdrop-filter: blur(12px);
  background: rgba(250, 250, 250, 0.8);
}
```

### 底部导航

```scss
.bottom-nav {
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
  z-index: 50;
  padding: 16px 24px;
  padding-bottom: calc(32px + var(--safe-area-bottom));
  background: rgba(250, 250, 250, 0.8);
  backdrop-filter: blur(20px);
  box-shadow: 0 -2px 16px rgba(17, 153, 142, 0.04);
}
```

### 卡片

```scss
.card {
  background: var(--surface);

  padding: 24px;
  box-shadow: 0 4px 20px rgba(17, 153, 142, 0.08);
```

## 动效规范

### 动画时长

| 类型 | 时长 | 使用场景 |
|------|------|----------|
| fast | 0.15s | 快速反馈、hover 状态 |
| normal | 0.3s | 标准过渡、页面切换 |
| slow | 0.45s | 慢速动画、入场效果 |

### 缓动函数

```scss
$ease-out: cubic-bezier(0.32, 0.72, 0, 1);      // 标准缓出
$ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);     // 双向缓动
$ease-spring: cubic-bezier(0.175, 0.885, 0.32, 1.275); // 弹性效果
```

### 关键帧动画

```scss
// 页面进入
```

## 响应式适配

### 安全区域

```scss
// 状态栏高度
--status-bar-height: 20px;

// 安全区域
--safe-area-top: 0px;
--safe-area-bottom: 0px;

// 导航栏高度
--nav-bar-height: calc(var(--status-bar-height) + 44px);
```

### 适配方案

1. 使用 CSS 变量动态设置安全区域
2. 底部固定元素使用 `calc(32px + var(--safe-area-bottom))`
3. 顶部导航栏使用 `padding-top: var(--status-bar-height)`
